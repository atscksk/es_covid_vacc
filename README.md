# es_covid_vacc

## 차트 및 지도, 검색 테이블이 포함된 전세계 COVID-19 백신 접종 현황 데이터 시각화 페이지
- 차트를 활용한 집계 데이터 시각화 페이지
- 세계 지도를 활용한 국가별 백신 접종 현황 시각화 페이지
- 테이블 페이지를 통한 일일 접종 기록 데이터 검색 기능
